Assignment Instructions:

Complete the PHP program named project1.php to determine whether a donor’s blood type is compatible with a recipient’s blood type.
The program should prompt the user for a donor blood type, then prompt the user for a recipient blood type and print out one of the
following four lines exactly:

    Compatible
    Incompatible
    ERROR: invalid donor blood type
    ERROR: invalid recipient blood type

Valid blood types are: O-, O+, A-, A+, B-, B+, AB-, and AB+.

Note: use the readline function to prompt the user for input.
